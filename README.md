# SOLID

SOLID is an acronym for five design principles intended to make software designs more understandable, flexible, and maintainable. Learn more about each principle in detail [here](solid-principles.md).

**S - Single Responsibility Principle (SRP)**

**O - Open-Close Principle (OCP)**

**L - Liskov Substitution Principle (LSP)**

**I - Interface Segregation Principle (ISP)**

**D - Dependency Injection Principle (DIP)**
